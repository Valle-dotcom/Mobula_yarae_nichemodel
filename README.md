# Ejercicio de modelado de nicho de *Mobula yarae*

Este repositorio contiene las instrucciones y archivos necesarios para llevar a cabo un ejercicio de modelado de nicho de 
La mantarraya del Atlántico (*Mobula yarae*) como parte del módulo de ecología del nivel 2 del [Taller de "Estudio de los patrones ecológicos y de divergencia genética de especies marinas en México"](https://www.fciencias.unam.mx/docencia/horarios/presentacion/371071) de la Facultad de Ciencias de la UNAM.
